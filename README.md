Student project

메뉴에 적힌대로 1번은 학생을 추가하는 메뉴, 2번은 학생의 정보를 갱신하는 메뉴, 
3번은 학생의 정보를 삭제하는 메뉴, 4번은 학생의 정보를 보여주는 메뉴입니다.

학생의 정보를 갱신하거나, 삭제하거나, 보여지길 원할 때는 해당학생의 학번을 입력해야 실행이 가능합니다.

먼저 1번을 입력하여 학생 정보를 추가하게 되면, stu_info라는 텍스트 파일에 입력한 정보가 등록됩니다.

2번을 누르면 수정하길 원하는 학생의 학번을 입력해야하고, 해당 학생이 존재한다면 새로운 핸드폰번호를 입력하여 정보를 갱신합니다.

3번을 누르면 삭제하길 원하는 학생의 학번을 입력해야하고, 해당 학생이 존재한다면 그 학생의 정보는 지워집니다.

4번을 누르면 화면으로 띄울 학생의 학번을 입력해야하고, 해당 학생이 존재한다면 그 학생의 정보가 화면에 출력됩니다.
